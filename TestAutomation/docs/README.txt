PROJECT TITLE: Sakai 11 LMS Testing
PURPOSE OF PROJECT: To create an automated testing framework for an FOSS
VERSION or DATE: 11/30/2016
HOW TO START THIS PROJECT: Clone the repository onto a machine running Linux (preferably Ubunutu 14.04). 
AUTHORS: James Rhoten, Max Hyaska, Joe Lester
USER INSTRUCTIONS: 
1)Ensure Java 1.8 is installed on your machine using the command 
  "java -version".
2)Navigate to the TestAutomation/scripts dir in the shell and run the command 
  "chmod +x runAllScripts.sh". 
3)Navigate up a directory to ./TestAutomation and run the script by typing 
  "./scripts/runAllScripts.sh".
